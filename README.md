# Centimental Support

**Centimental** is a chromatic tuner for guitar, bass, and ukulele with 40+ visual themes, custom tunings, capo support, and an Apple Watch app. This site is where support, documentation, and policies for the app live.

## Documentation

- **[User Guide](user-guide.md)**: full feature walkthrough covering getting started, gestures, string indicators, themes, Apple Watch, Pro features, and more.
- **[Troubleshooting](troubleshooting.md)**: common questions and fixes.
- **[Theme Gallery](themes/)**: screenshots of every visual theme at different tuning states.
- **[Changelog](changelog.md)**: what's new in each release.

## About

Centimental is an independent app by Ed Caspersen. The tuner engine is built on [AudioKit](https://audiokit.io/) with a custom YIN-based pitch-detection pipeline tuned for guitar and bass. Themes are hand-designed SwiftUI views.
