# Theme Gallery

Centimental ships with 40+ hand-designed visual themes. Each one renders the same precise pitch-detection data in a different visual language — pick whichever matches your mood.

Every theme is shown at three tuning states: **−35¢** (flat), **0¢** (in tune), and **+35¢** (sharp).
